# Sycophant
### A very simple discord bot that agrees with everything you say.

I do not know how to code. XOXO

This bot adds the following slash commands:

/togglereacts will make the bot add positive reactions to your own messages.
/togglereplies will make the bot reply positively to your own messages.
/togglehatereact will make the bot add negative reactions to messages that you react to with the "👎" emoji.
/togglehatereply will make the bot reply negatively to messages that you react to with the "👎" emoji.
/toggleglazereact will make the bot add positive reactions to messages that you react to with the "🔥" emoji.
/toggleglazereply will make the bot reply positively (with the same set of responses as "/togglereplies") to messages that you react to with the "🔥" emoji.
